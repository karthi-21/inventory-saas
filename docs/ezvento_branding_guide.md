# Ezvento Branding Guide

Ezvento is a simple, powerful SaaS product built for Indian retail businesses. The brand should feel trustworthy, calm, and practical while still conveying capability and scale.

## Brand Essence

Ezvento exists to simplify how businesses operate. It should feel like a dependable partner that removes complexity rather than adding to it.

Core idea:
Simple on the surface, powerful underneath.

Tagline:
Simplifying how you run your store

## Target Users

Primary:
- Small shop owners (kirana, pharmacy, local retail)
- Semi-technical users

Secondary:
- Growing retail stores
- Multi-store operators

The brand should feel approachable for beginners and reliable for professionals.

## Brand Personality

- Simple
- Trustworthy
- Calm
- Slightly premium
- Supportive (not robotic)

Avoid:
- Overly playful tone
- Complex or technical language
- Aggressive startup-style branding

## Tone of Voice

Base tone:
- Short
- Clear
- Direct

Examples:
- Add item
- Enter price
- Total ₹500

Assistant tone (only when helpful):
- This item is low in stock
- You sold 20 today
- Do you want to reorder?

Error tone:
- Something went wrong. Try again.

Avoid:
- Long sentences
- Jargon
- Confusing instructions

## Visual Direction

Primary Color:
- Blue (trust, reliability)

Secondary Color:
- Green (money, growth)

Accent:
- Orange (actions, highlights)

Usage:
- Blue dominates UI
- Green for positive indicators
- Orange for buttons and important actions

Keep overall UI clean and minimal.

## Typography

- Highly readable
- Clean and modern
- Slightly rounded for friendliness

Avoid decorative fonts.

## Logo Direction

- Simple wordmark: "Ezvento"
- Optional icon:
  - Box (inventory)
  - Receipt (billing)
  - Subtle “E” shape

Style:
- Flat
- Minimal
- Scalable for app icon

## Product Naming

Always use:
Ezvento

Then extend:
- Ezvento POS
- Ezvento Inventory
- Ezvento Reports

Avoid:
- Over-complicating names
- Creating unrelated sub-brands

## UX Writing Principles

- Use common words:
  - Stock instead of Inventory
  - Sales instead of Transactions
- Keep actions obvious
- Reduce thinking for the user

## Emotional Layer

The product should feel like:
- A helpful assistant
- A reliable system
- A daily business tool

Not:
- A complex software
- A technical system

## Long-Term Direction

- Start India-first
- Expand globally
- Add regional language support later

Keep branding neutral enough to scale beyond local markets.

## Summary

Ezvento should feel:
- Easy to use
- Reliable every day
- Quietly powerful

It is not just a POS system.
It is a simple business operating system for retail.
